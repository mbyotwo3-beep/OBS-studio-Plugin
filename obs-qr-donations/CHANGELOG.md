# Changelog

All notable changes to the OBS QR Donations plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]
### Added
- Initial release of the OBS QR Donations plugin
- Support for multiple cryptocurrencies (BTC, ETH, LTC, etc.)
- Lightning Network integration via Breez SDK
- Real-time payment notifications
- Customizable QR code appearance
- Streamer-focused features (auto-copy on stream start)
- Comprehensive API documentation
- Test suite for Lightning Network integration
- Installer scripts for Windows, macOS, and Linux
- Screenshot and tutorial generation tools

### Changed
- Improved error handling and logging
- Enhanced UI/UX for better user experience
- Optimized performance for streaming environments
- Updated documentation with detailed setup instructions

### Fixed
- Fixed issues with QR code generation
- Resolved memory leaks in payment processing
- Addressed thread safety concerns
- Fixed compatibility issues with OBS Studio 28.0+

## [1.0.0] - 2023-11-19
### Added
- Initial release of the OBS QR Donations plugin
- Basic QR code generation for multiple cryptocurrencies
- Simple configuration interface
- Basic documentation
